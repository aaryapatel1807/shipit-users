How to Contribute to Open Source ðŸš€

Follow these steps to fork, commit, and open a Pull Request like a pro (and not annoy maintainers):

  1. Fork & Clone 
    git clone git@github.com:YOUR-USERNAME/REPO.git
cd REPO
  2. Add Upstream 
    git remote add upstream git@github.com:ORIGINAL-OWNER/REPO.git
git fetch upstream
git checkout main
git merge upstream/main
  3. Create a Branch 
    git checkout -b fix/issue-123-description
  4. Make Changes & Test 
    # run tests depending on project
npm test
pytest -q
  5. Commit 
    git add .
git commit -m "Fix: handle null avatar (issue #123)"
  6. Push 
    git push origin fix/issue-123-description
  7. Open Pull Request 
    Title: Fix: handle null avatar (#123)
Body:
- What changed
- Why it changed
- How to test
